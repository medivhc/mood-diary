# 📔 心情日记 Telegram Web App

> 记录你的每一天 - 一个好玩的心情追踪应用

## 🎉 项目状态

✅ **已上线！** 立即体验：

- 🌐 Web App URL: **https://medivhc.github.io/mood-diary/**
- 🤖 Telegram Bot: **@medivhc_ai_bot**
- 📱 访问方式：打开 Bot，点击底部菜单按钮「📔 写日记」

---

## ✨ 功能特色

### 1. ✍️ 写日记
- 🎨 5 种心情表情（🤩😊😐😔😢）
- 📝 自由记录当天事件和感受
- 💾 自动保存到本地存储

### 2. 📊 情绪统计
- 📈 心情趋势图表（Chart.js）
- 🤖 AI 情绪关键词分析
- 📅 记录天数、平均心情、连续记录统计

### 3. 📜 历史记录
- 📖 按时间倒序查看所有日记
- 🔍 快速回顾过往心情
- 📊 每条日记附带 AI 分析

### 4. 🏆 成就系统
| 徽章 | 名称 | 解锁条件 |
|------|------|---------|
| 🌱 | 第一次记录 | 写第一篇日记 |
| 🌿 | 三日新人 | 累计 3 篇日记 |
| 🔥 | 周记达人 | 累计 7 篇日记 |
| 🏆 | 月度坚持 | 累计 30 篇日记 |
| ⭐ | 开心时刻 | 记录开心心情 (≥4) |
| 💙 | 真实面对 | 记录低落心情 (≤2) |
| ✍️ | 文字达人 | 单篇 100+ 字 |
| 🦉 | 深夜思考 | 23 点后记录 |

---

## 🎨 设计亮点

- **渐变紫色主题** - 现代美观的视觉效果
- **完美移动端适配** - 针对手机屏幕优化
- **流畅动画效果** - 心情选择、Tab 切换、成就解锁
- **Telegram 深度集成** - WebApp SDK、主题适配
- **隐私保护** - 所有数据存储在本地（localStorage）

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| HTML5/CSS3 | 页面结构和样式 |
| JavaScript (原生) | 交互逻辑 |
| Chart.js | 心情趋势图表 |
